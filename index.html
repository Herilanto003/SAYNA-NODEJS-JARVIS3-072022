VOICI TOUS LES ROUTES DE L' API

On decrit ici tous les routes de l' api, on a classé l' api selon leur route pour faciliter le travail de récuperation

ROUTE METHODE HEADERS DESCRIPTION
/ GET {} Affichage de la page index.html
/saynajarvis/login POST {} Route d'authentification d'un utilisateur
/saynajarvis/register POST {} Route d'insciption d'un utilisateur
/saynajarvis/invitation POST { "Authorization": "Bearer xxxxxx" } Route invité un utilisateur
/saynajarvis/invitation?token=xxx&resp=xxx GET { "Authorization": "Bearer xxxxxx" } Route gestion d'une invitation
/saynajarvis/room POST { "Authorization": "Bearer xxxxxx" } Route de création d'un piece
/saynajarvis/room?name=xxxx DELETE { "Authorization": "Bearer xxxxxx" } Route de supression d'un piece
/saynajarvis/object POST { "Authorization": "Bearer xxxxxx" } Route de création d'un object
/saynajarvis/object?name=xxx DELETE { "Authorization": "Bearer xxxxxx" } Route de supression d'un object
/saynajarvis/object PUT { "Authorization": "Bearer xxxxxx" } Route de changement d'un object
/saynajarvis/user?name=xxxx DELETE { "Authorization": "Bearer xxxxxx" } Route de suppression du compte
